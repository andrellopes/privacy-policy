Política de Privacidade

Última atualização: 26 de junho de 2025

O aplicativo HoraCerta+ foi desenvolvido com o objetivo de exibir a hora atual de forma simples e clara. Valorizamos a sua privacidade e queremos deixar claro como lidamos com os dados dos usuários.

Coleta de Dados

O aplicativo não coleta, armazena nem compartilha dados pessoais de nenhum usuário. Nenhuma informação de localização, câmera, microfone, contatos ou quaisquer dados sensíveis são acessados.

Anúncios

Este aplicativo utiliza o Google AdMob para exibição de anúncios. O AdMob pode coletar dados não pessoais para fins de personalização de anúncios conforme as políticas do Google.

Para mais informações, consulte a Política de Privacidade de Anúncios do Google.

Público-alvo

O HoraCerta+ é adequado para todas as idades. Ele não contém conteúdo impróprio nem solicita dados pessoais de crianças ou adultos.

Alterações

Esta política pode ser atualizada no futuro. Recomendamos que os usuários consultem esta página periodicamente.
